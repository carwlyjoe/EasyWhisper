
## 使用方法

1. 双击运行 `启动.bat`
2. 在打开的图形界面中：
   - 选择要转录的音频或视频文件
   - 选择已下载的模型文件
   - 设置输出文件的保存位置和格式
   - 点击"开始转录"
3. 等待进度条完成，即可在指定位置找到转录结果

## 支持的格式

### 输入格式
- 音频：WAV、MP3、M4A、FLAC
- 视频：MP4、AVI、MKV、MOV

### 输出格式
- SRT 字幕文件
- 纯文本文件 (TXT)

## 注意事项

- 确保系统已正确配置 FFmpeg 环境变量
- 首次使用时需下载语音模型文件
- 转录速度取决于选择的模型大小和计算机性能
